# Corner Brand Book

Complete brand identity guidelines for Corner.

---

## 1. Brand Overview

**Corner** is a B2B digital solutions business helping Algerian SMEs become more professional, visible, and accessible.

### Brand Personality

- **Professional** — We are serious about business
- **Bold** — We stand out with confidence
- **Simple** — We make things easy
- **Trustworthy** — We deliver what we promise
- **Local** — We understand Algerian businesses

---

## 2. Logo

### Logo Concept

The Corner logo represents:

- A **corner** (meeting point, central location)
- **Simplicity** and **clarity**
- **Professionalism** for local businesses

### Logo Variations

#### Primary Logo

- Full Corner wordmark
- Black text + Yellow accent
- Use on white/light backgrounds
- **File:** `logo/logo-primary.svg`

#### White Logo

- All white version
- Use on dark backgrounds or photos
- **File:** `logo/logo-white.svg`

#### Black Logo

- All black version
- Use when color is not available (print, fax, photocopies)
- **File:** `logo/logo-black.svg`

#### Icon/Mark

- Simplified icon version
- Use for social media profile pictures, favicons, small spaces
- **File:** `logo/logo-icon.svg`

### Logo Usage Rules

✅ **Do:**

- Use official logo files only
- Maintain clear space around logo (minimum = height of "C")
- Use on clean, uncluttered backgrounds
- Keep proportions intact

❌ **Don't:**

- Stretch or distort the logo
- Change colors (except approved variations)
- Add effects (shadows, gradients, outlines)
- Rotate the logo
- Place on busy backgrounds without background treatment

See `logo/logo-usage-rules. md` for detailed examples.

## 3. Color Palette

Corner uses a **bold, minimal** color palette.

### Primary Colors

#### Yellow

```
Hex: #FFCE18
RGB: 255, 206, 24
Usage: Primary brand color, CTAs, highlights, QR codes
```

#### Black

```
Hex: #000000
RGB: 0, 0, 0
Usage: Text, headings, UI elements
```

#### White

```
Hex: #FFFFFF
RGB:  255, 255, 255
Usage:  Backgrounds, negative space, contrast
```

### Color Usage

**Yellow** — Use for:

- Call-to-action buttons
- Highlights and emphasis
- QR code backgrounds
- Icons and illustrations accents
- Links (hover state)

**Black** — Use for:

- Body text
- Headings
- UI borders
- Backgrounds (when using white text)

**White** — Use for:

- Page backgrounds
- Card backgrounds
- Text on dark backgrounds

### Color Combinations

✅ **Recommended:**

- Yellow background + Black text
- Black background + White text
- White background + Black text + Yellow accents
- Yellow background + White text (only for large text/CTAs)

❌ **Avoid:**

- Black text on yellow (body text — hard to read)
- Grey tones (use black or white instead)

## 4. Typography

Corner uses **Helvetica** for both Latin and Arabic scripts.

### Typeface

**Helvetica** (or Helvetica Neue)

**Why Helvetica? **

- Professional and timeless
- Highly readable
- Works in Arabic and Latin
- Widely available
- Clean and neutral

### Font Weights

- **Regular (400)** — Body text
- **Medium (500)** — Subheadings, emphasis (optional)
- **Bold (700)** — Headings, CTAs

### Typography Scale

```
H1 — Bold, 48px / 3rem (Desktop), 32px / 2rem (Mobile)
H2 — Bold, 36px / 2.25rem (Desktop), 28px / 1.75rem (Mobile)
H3 — Bold, 28px / 1.75rem (Desktop), 24px / 1.5rem (Mobile)
H4 — Bold, 24px / 1.5rem
H5 — Bold, 20px / 1.25rem
Body — Regular, 16px / 1rem
Small — Regular, 14px / 0.875rem
```

### Line Height

- **Headings:** 1.2
- **Body text:** 1.6
- **UI elements:** 1.4

### Letter Spacing

- **Headings:** -0.02em (tight)
- **Body:** 0 (normal)
- **Buttons/CTAs:** 0.02em (loose)

### Arabic Typography Notes

- Use **Helvetica Arabic** (if available) or system Arabic fonts
- Maintain same hierarchy as Latin
- Right-to-left (RTL) layout
- Keep generous line-height for readability

## 5. Visual Style

### Design Principles

1. **Bold & Clear**

   - Strong contrast (black/white/yellow)
   - Clear hierarchy
   - Generous white space

2. **Simple & Direct**

   - No unnecessary decoration
   - Functional design
   - Clean layouts

3. **Professional**
   - Polished and modern
   - Trustworthy presentation
   - Quality over quantity

### Graphic Elements

#### Shapes

- Clean geometric shapes
- Right angles and corners (brand theme)
- Minimal rounded corners (4px - 8px max)

#### Icons

- Outlined style (2px stroke)
- Black or yellow
- Simple and recognizable
- See `/assets/icons/`

#### Photography

- Real Algerian businesses
- Clean, well-lit
- Professional but approachable
- Avoid stock photos when possible

#### Illustrations

- Simple, geometric
- Yellow + Black color scheme
- Support the message, don't distract
- See `/assets/illustrations/`
